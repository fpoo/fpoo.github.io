5. Impresión 3D

Impresión 3D

La impresión 3D, también llamada manufactura por adición (inglés), es un conjunto de procesos que producen objetos a través de la adición de material en capas que corresponden a las sucesivas secciones transversales de un modelo 3D. Los plásticos y las aleaciones de metal son los materiales más usados para impresión 3D, pero se puede utilizar casi cualquier cosa, desde hormigón (inglés) hasta tejido vivo.

El software que utilizaremos para impresión 3D será “Ultimaker Cura”.

Link de descarga del Software:

https://ultimaker.com/es/software/ultimaker-cura

Set Up de Cura

Una vez instalado el programa en nuestros computadores, debemos hacer el set up de la impresora. Vamos a ADD PRINTER en la barra superior y luego ADD A NON-NETWORKED PRINTER para trabajar offline. Dentro de la lista de impresoras escogemos la “Creality Ender 3”, que vendría a ser la que podemos ver en la foto.

Posteriormente, debemos escoger el material con el cual la Ender estará imprimiendo. Para esto, vamos al ícono amarillo donde dice “Generic PLA”. Se desplegará un menú para escoger el material y el tamaño del cabezal (Nozzle Size). En este caso en material escogemos eSUN y luego PLA+, el color puede ser cualquiera. Para este caso el cabezal que utilizaremos será de 0.4mm.

Posteriormente, vamos al ícono de carpeta para abrir nuestro modelo y lo ubicamos como más nos acomode utilizando la barra izquierda en el programa.

Si el producto que vamos a imprimir es curvo o irregular podremos ver algunas zonas en rojo. Esto significa que en esas partes se va a necesitar de “soporte”, es decir, de un relleno delgado y fácil de romper una vez impreso el objeto.

A pesar de esto, nos dejará una pequeña textura en nuestro objeto, entonces debemos ser cuidadosos y remodelar el objeto para tratar de ocupar la menor cantidad de soporte posible.

Para editar los valores del material de soporte, como el ángulo o el lugar debemos seleccionar el menu la opcion NOT SUPPORTED y luego SUPPORT

Para finalizar, apretamos SLICE y esperamos a que el programa nos genere las capas en el orden correcto que seguirá la impresora.

Podemos visualizar las capas con el slider en el costado derecho y simular el movimiento que seguirá el cabezal de la impresora 3D con el slider inferior, tal como se muestra en el video. Cura nos entrega también un tiempo aproximado de impresión, que en este caso sería de 3 horas y 13 minutos.

Podemos saber también el significado respectivo de cada color en nuestro al desplegar el menú COLOR SCHEME

Tiempo de Impresión

Podrán notar que 3 horas es mucho tiempo para un objeto tan pequeño como este, a pesar de su complejidad geométrica, sin embargo, esto se pude modificar para que sea menor.

Para modificar el tiempo debemor ir al menú de ajustes de impresión (NOT SUPPORTED) y luego le damos a SUPPORT. Dentro de esta categoría está el ángulo en el cual se posicionará el material de soporte (“Support Overhang Angle”), en este caso, meintras mayor sea el ángulo, menor será el tiempo de impresión.

En este caso, el ángulo anterior era de 60º y se aumentó a 70º, esto provocó una disminución de 24 minutos de impresión (manteniendo el resto de los ajustes iguales)

Otra opción que sirve para disminuir el tiempo de impresión es el INFILL. En esta opción modificamos el número en “Infill Density”, en este caso, mientras menor sea la densidad con la cual se va a rellenar el interior de las paredes de nuestro objeto, menor será el tiempo de impreisón.

En este caso, se cambió un infill de 20% a 15% y el tiempo de impresión disminuyó en 30 minutos.
